# Purpose

Deploy Flow Finance AI on an existing omniflow Debian host where Firefly III (`firefly`), PostgreSQL (`postgres`), and Traefik already share the Docker network **`traefik`**. This guide covers operator steps for US-0010 — no bundled `firefly-iii` or `postgres` containers from this repo.

# Prerequisites

- Docker Compose **v2.24+** (`!reset` merge support)
- Host stacks running on network `traefik`:
  - Firefly container `firefly` (public UI `https://finance.omniflow.cc`)
  - PostgreSQL container `postgres` with database `flow_finance_ai` and TimescaleDB extension
  - Traefik with middleware `auth` and cert resolver `myresolver`
- Firefly Personal Access Token (PAT) with read access
- No access to host Traefik credential files from this repo — basic-auth is configured on the host Traefik stack only

# Usage steps

## 1. TimescaleDB preflight

On shared host Postgres, create database and role (once):

```sql
CREATE DATABASE flow_finance_ai;
CREATE USER finance WITH PASSWORD '<operator-secret>';
GRANT ALL ON DATABASE flow_finance_ai TO finance;
\c flow_finance_ai
CREATE EXTENSION IF NOT EXISTS timescaledb CASCADE;
SELECT extversion FROM pg_extension WHERE extname = 'timescaledb';
```

**Pass:** non-null `extversion`. **Fail:** install TimescaleDB on host Postgres before deploy.

## 2. Configure `.env`

Copy `.env.example` to `.env` on the deploy host. Required for external profile:

| Variable | Value |
|----------|-------|
| `COMPOSE_FILE` | `docker-compose.yml:docker-compose.external.yml` |
| `COMPOSE_PROFILES` | `external` |
| `DATABASE_HOST` | `postgres` (overlay default) |
| `DATABASE_PASSWORD` | operator secret |
| `FIREFLY_BASE_URL` | `http://firefly:8080` |
| `FIREFLY_PERSONAL_ACCESS_TOKEN` | Firefly PAT |
| `GRAFANA_ADMIN_PASSWORD` | replace default `admin` |

**Not required for external:** `FIREFLY_APP_KEY`, `FIREFLY_DB_*` (only for `bundled-firefly` greenfield profile).

Optional overrides:

| Variable | Default | Purpose |
|----------|---------|---------|
| `TRAEFIK_HOST` | `financegnome.omniflow.cc` | Public app hostname |
| `TRAEFIK_MIDDLEWARE` | `auth` | Host Traefik basic-auth middleware name |
| `GRAFANA_TRAEFIK_HOST` | empty | Set only to expose Grafana publicly |

## 3. Deploy

**Profile rule:** use **`external` only**. Never combine with `minimal`, `standard`, `full`, or `bundled-firefly`.

```bash
docker compose -f docker-compose.yml -f docker-compose.external.yml --profile external up -d --build
```

Expected project services: **`flow-finance-ai`**, **`grafana`** only.

Verify:

```bash
docker compose -f docker-compose.yml -f docker-compose.external.yml --profile external ps --services
```

## 4. Public URL and edge auth

- App URL: `https://financegnome.omniflow.cc`
- Traefik `auth` middleware (basic-auth) protects the edge — credentials live on host Traefik stack, not in this repo
- Backend health: `https://financegnome.omniflow.cc/health` (with basic-auth) should return OK when DB + PAT are configured

## 5. Grafana access (internal default)

External overlay does **not** publish Grafana on a host port or public Traefik route by default.

Access options:

- From a container on `traefik` network: `curl http://grafana:3000/login`
- SSH to host and use Docker exec/network tools as documented in runbook

Optional public host: set `GRAFANA_TRAEFIK_HOST` (e.g. `grafana-financegnome.omniflow.cc`) before deploy.

## 6. OIDC registration (when auth enabled)

Register with your IdP (not automated):

| Setting | Value |
|---------|-------|
| Redirect URI | `https://financegnome.omniflow.cc/callback` |
| Post-logout | `https://financegnome.omniflow.cc/` |
| Origin | `https://financegnome.omniflow.cc` |

Set `VITE_OIDC_*` and `OIDC_*` in `.env`. For smoke testing, `AUTH_DEV_BYPASS=true` is acceptable; production requires IdP registration.

Traefik basic-auth and app OIDC are separate layers.

# Troubleshooting

| Issue | Fix |
|-------|-----|
| Migration panic / health never OK | Run TimescaleDB preflight on `flow_finance_ai` |
| Duplicate Firefly containers | Do not combine `external` with `bundled-firefly` or `minimal` |
| `curl http://firefly:8080` fails from app container | Verify host `firefly` on `traefik` network |
| `/health` 503 | Check `DATABASE_*`, TimescaleDB extension, PAT |
| TLS route 401 without credentials | Expected — Traefik `auth` middleware |
| Sync/API 401 with auth enabled | Configure OIDC or use `AUTH_DEV_BYPASS` for smoke only |
| Grafana unreachable | Use traefik-network curl; overlay clears host port publish |
