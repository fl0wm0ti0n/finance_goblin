# US-0019 — Goal-driven planning

## Purpose

Create **goal balance** plans with a target amount and date, view **per-plan** progress stats, apply **category** spending reductions (capped by mirror history), and review deterministic savings suggestions — without changing household Plan vs Actual scope (DEC-0096).

## Create a goal plan (`/planning` → Scenarios)

1. Enter a plan name (empty-state onboarding) or use **Create goal plan** on the Goal balance card.
2. Set **Target €**, **target date** (today or later), and optional **goal account id** (Firefly asset account).
3. When account is omitted, the backend defaults to the max-balance asset in reporting currency (DEC-0095).
4. The plan appears in the Scenarios list with template `goal_balance`.

## Goal stats strip (AC-2)

When a goal plan is selected:

- **Scenarios** and **Compare** show the goal stats strip for **this plan only** — monthly delta vs baseline, calendar-year rollup, projected balance at target.
- If the target date is beyond the 730-day projection horizon, projected balance is omitted with horizon copy (DEC-0092).
- Off-track plans show gap € and required monthly savings (0% interest; informational only — no auto-lines).
- **Plan vs Actual** remains the **active household** plan (DEC-0096).

## Category overlay cap (AC-3)

- `remove_outflow` on **category** targets is capped at the 3-month average mirror outflow (DEC-0093).
- Empty category history → zero overlay effect.
- `add_outflow` category lines use the full amount on the household schedule.

## Category savings modal (AC-4 / AC-5)

1. On a goal plan, open **Category savings suggestions**.
2. Review ranked categories (aggregates only — no raw transactions).
3. Checkbox-select lines and **Apply selected** to create `remove_outflow` category adjustments.
4. Confirm triggers recompute; nothing is auto-applied.

Optional: AI chat tool `get_category_savings` returns the same ranking (DEC-0097).

## Troubleshooting

| Symptom | Likely cause | Action |
|---------|--------------|--------|
| 422 on create | Missing target fields or past date | Set balance + future `target_date` |
| 404 on goal-stats | Plan is not `goal_balance` | Create or select a goal template plan |
| No savings suggestions | Spend below €20/mo or fixed bucket | Widen months or pick another category |
| Compare vs goal stats differ | Compare is household; goal uses account scope | Documented per DEC-0095 |
