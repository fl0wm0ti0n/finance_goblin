# DEC-0104 — Axum SPA index.html fallback (BUG-0016)

**Status:** Accepted  
**Date:** 2026-06-09  
**Work item:** BUG-0016 (acceptance **AX**)  
**Research:** [R-0086](../docs/engineering/research.md#r-0086--bug-0016-spa-deep-link-fallback-axum-vs-traefik)  
**Extends:** **DEC-0057** (Grafana proxy route ordering — proxy before SPA fallback)  
**Supersedes:** BUG-0009 analytics 404 advisory for unauthenticated curl on `/analytics/{slug}` (shell routing only; panel data gate unchanged)

## Context

React Router `BrowserRouter` deep links (`/forecast`, `/analytics/cashflow`, etc.) require the server to return the SPA shell (`index.html`) for paths that are not static files. `build_router` currently uses plain `ServeDir` as the fallback, which returns **HTTP 404** when no file exists at the request path. Root `/` works; client-side navigation works after initial load. Traefik on omniflow forwards paths unchanged — fixing Axum fixes all deploy profiles.

## Options

| Option | Verdict | Rationale |
|--------|---------|-----------|
| **A — Axum `ServeDir::fallback(ServeFile::new(index.html))`** | **Accepted** | Returns **HTTP 200** per acceptance **AX**; single change covers Docker (`/app/static`), local override (`:18080`), and omniflow |
| **B — `ServeDir::not_found_service(ServeFile)`** | **Rejected** | Returns **404** with HTML body — fails curl 200 gate |
| **C — Traefik catch-all / nginx sidecar** | **Rejected** | Duplicate responsibility; Traefik is reverse-proxy only (R-0086 §2) |
| **D — Backend `/callback` redirect handler** | **Rejected** | OIDC token exchange is React-only (`OidcCallback`); redirect breaks bookmarked paths |
| **E — `axum_extra::routing::SpaRouter`** | **Deferred** | Vite emits `/assets/*` at root; `ServeDir` + `fallback` is simpler |

## Decision

Accept **DEC-0104** as the Axum-only SPA fallback contract:

| Contract element | Value |
|------------------|-------|
| Placement | `backend/src/lib.rs` — `build_router` only; **no Traefik label change** for MVP |
| Static dirs | Production: `/app/static`; dev parity: `frontend/dist` when present |
| Fallback API | `ServeDir::new(dir).fallback(ServeFile::new(dir.join("index.html")))` |
| HTTP status | **200** for missing non-API paths (not 404-with-body) |
| Route merge order | `health` → `analytics::grafana_routes` (**DEC-0057**) → `api::routes` → SPA fallback (unchanged) |
| `/callback` | No backend route; SPA fallback serves `index.html`; React handles OIDC |
| Protected prefixes | `/api/v1/*` JSON responses preserved; `/analytics/grafana/*` proxy preserved; hashed `/assets/*` served when present |
| Tests | `build_router` integration test (deep link 200 + API non-HTML); QA curl matrix per R-0086 §6 |

### Implementation sketch

```rust
use tower_http::services::{ServeDir, ServeFile};

let index = static_dir.join("index.html");
let spa = ServeDir::new(static_dir).fallback(ServeFile::new(index));
router = router.fallback_service(spa);
```

Apply identically to the `frontend/dist` branch.

## Risks

| Risk | Mitigation |
|------|------------|
| API paths receive `index.html` | Preserve merge order; integration test `GET /api/v1/…` ≠ HTML |
| Grafana proxy swallowed by SPA | `grafana_routes` merged before fallback; test `/analytics/grafana/…` |
| `index.html` missing in image | Dockerfile copies `dist` → `/app/static`; optional startup fail-fast |
| Traefik `/analytics/*` no-auth exposes SPA slug | Pre-existing US-0011 tradeoff; HTML shell only — API JWT/OIDC gated |
| OIDC `/callback` broken by redirect middleware | Do not add `Redirect` fallback |
