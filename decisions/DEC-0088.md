# DEC-0088 — CategoryFilter and bar trend chart UX (US-0018)

**Status:** Accepted  
**Date:** 2026-06-08  
**Story:** US-0018  
**Research:** [R-0083 §2, §4](research.md#r-0083--us-0018-category-filters-expense-series-api--trend-analytics)

## Context

AC-1 requires a shared category filter on multiple surfaces; AC-3/AC-4 require a trend chart with month labels and performance insight. Discovery recommended single-category MVP and bar default; research aligned with existing `MonthlyChart` (ECharts bars).

## Decision

### 1. CategoryFilter component (frozen)

- **Single-select** MVP; multi-category overlay **deferred** (stretch: ≤3 series)
- Shared React component: `frontend/src/components/category/CategoryFilter.tsx`
- Data: `GET /api/v1/categories`
- Sentinel `""` = All categories (toolbar contexts)
- Explicit `__uncategorized__` option in dropdown
- ≤20 categories: `<select>` pattern (clone Forecast account picker); >20: searchable combobox without new dependency

### 2. CategoryTrendChart component (frozen)

- **Bar chart default** (ECharts) — discrete month EUR labels per AC-3
- Data: `GET /api/v1/categories/expense-series`
- Requires explicit category selection (disabled / prompt when none)
- Empty state: "No categorized spending in this period"
- MoM / best / worst: render from API `summary` (callouts or table — AC-4)
- **Line toggle:** stretch P2 only — not MVP

### 3. Primary home

`/forecast` Monthly tab — filter above stat cards; chart below household `MonthlyChart`.

Component exported for Planning and Wealth reuse.

## Alternatives rejected

| Alternative | Why rejected |
|-------------|--------------|
| Multi-series overlay (≤3) | Scope creep; AC-3 satisfied by single series |
| Line chart default | AC-3/AC-4 magnitude comparison favors bars (R-0083 §4) |
| Dedicated `/analytics/category-trends` only | Discovery primary home is forecast monthly |
| New chart library | ECharts already in stack |

## Risks

- All-categories sentinel on trend chart could mislead → require explicit pick for chart
- Long category lists without search → combobox threshold at 20
