# DEC-0087 — Category expense-series API and uncategorized sentinel (US-0018)

**Status:** Accepted  
**Date:** 2026-06-08  
**Story:** US-0018  
**Research:** [R-0083 §1–2](research.md#r-0083--us-0018-category-filters-expense-series-api--trend-analytics)  
**Extends:** BUG-0006 mirror ingest, R-0060 aggregate contract, `label_uncategorized_categories`

## Context

US-0018 AC-2 requires a monthly per-category expense API; AC-5 requires explicit uncategorized handling. Mirror has period `aggregates_by_category` only — no monthly-by-category dimension. Discovery drafted endpoints; research validated month-spine SQL and `__uncategorized__` token.

## Decision

### 1. New repository method

`TransactionsRepository::expense_series_by_month(category_id, start, end)` using:

- `generate_series` month spine from computed window (not data min/max)
- `LEFT JOIN transactions` on `date_trunc('month', t.date)`
- Outflow: `ABS(amount)` where `amount < 0`; inflow where `amount > 0`
- Full spine months with explicit €0 when no transactions

### 2. REST endpoints

| Endpoint | Contract |
|----------|----------|
| `GET /api/v1/categories` | Mirror catalog `{id, name}`; sort by name; `?q=` with `MIN_CATEGORY_SEARCH_LEN=2`; cap 200 + `truncated` flag |
| `GET /api/v1/categories/expense-series` | Required `category_id`; `months` default 12 max 24; optional `end` default today |

### 3. Uncategorized sentinel (frozen)

Query token `category_id=__uncategorized__` maps to `t.category_id IS NULL`.

Response includes `uncategorized: true` and `category_label: "Uncategorized"`. Never silent zero without bucket metadata.

### 4. Summary object (AC-4)

Server-computed on spine:

- `mom_delta_pct` — last two calendar months in window (zeros allowed)
- `best_month` / `worst_month` — by `outflow_eur`

### 5. Privacy

Aggregate-only responses; no raw transaction rows (DEC-0032 posture).

## Alternatives rejected

| Alternative | Why rejected |
|-------------|--------------|
| Gap-fill only months with data | Fails AC-3 month-label contract |
| Materialized view / continuous aggregate | Overkill for ≤24 points on ~1k rows |
| Separate `/uncategorized` route | Extra surface; sentinel suffices |
| Data min/max window | Inconsistent month labels when category idle |

## Risks

- Unknown `category_id` after Firefly delete → 404 + empty-state guidance
- `date_trunc` may limit index use — mitigated by DEC-0090
- Mixed native currency edge cases — EUR mirror default; footnote deferred
