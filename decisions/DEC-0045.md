# Decision DEC-0045

**Status:** Accepted  
**Date:** 2026-06-02  
**Story:** US-0008  
**Research:** R-0038, R-0039, R-0041  
**Extends:** DEC-0031 orchestration loop, US-0006 `tool_choice: auto`

## Context

OpenAI accepts `tool_choice: "auto"` on every tool round. Ollama OpenAI-compat docs list **`tool_choice` unsupported** (R-0038). LM Studio/vLLM variance requires defensive defaults for mixed local endpoints (R-0039).

## Options

1. **Omit `tool_choice` when `AiProvider::omit_tool_choice()` is true** — recommended
2. **Always send `tool_choice: auto`** — rejected (Ollama may error or ignore unpredictably)
3. **Send `tool_choice: required` for local nudge** — rejected (unsupported on Ollama; R-0041)
4. **Separate orchestrator loops per provider** — rejected (violates AC4 frozen tool layer)

## Recommendation

- **`openai`:** `omit_tool_choice = false` — send `tool_choice: Some("auto")` (unchanged US-0006 behavior)
- **`ollama` and `openai_compatible`:** `omit_tool_choice = true` — **omit field** from JSON request body (`skip_serializing_if` already on struct)

Add optional TOML `[ai] temperature` with defaults:
- **OpenAI:** `0.7` (unchanged implicit behavior)
- **Local (`is_local`):** `0.3` — improves deterministic tool name selection (R-0041)

**Streaming + tools:** keep tool rounds **`stream: false`**; stream only final assistant pass (R-0029, R-0041) — unchanged US-0006 pattern, critical for LM Studio/Ollama delta accumulation.

**Recommended operator models (documented, not enforced):**
- Dev: `llama3.1:8b`
- Prod default: `qwen2.5:14b` (R-0038, R-0041)

## Impact

- Orchestrator builds `ChatCompletionRequest` using `provider.omit_tool_choice()` before each tool round
- Provider sets temperature on request when field added
- No change to six tool schemas or PrivacyLayer

## Risks

- Local models may still omit `tool_calls` despite tools array — handled by DEC-0046 fallback
- Future Ollama versions may add `tool_choice` support — preset flag remains correct defensive default
